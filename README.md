# J4RV's Advent of Code 2020

----------

## Execute Day's XX solution:

Getting the module:

```
go get -u github.com/j4rv/advent-of-code-2020
go run "github.com/j4rv/advent-of-code-2020/Day XX"
```

OR cloning the repo, and running from the root directory:

```
go run "github.com/j4rv/advent-of-code-2020/Day XX"
```
